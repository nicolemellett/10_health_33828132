# 10_Health_33828132

A fitness tracker web application built with Node.js, Express, and EJS .

This repository contains the source code for a health application, including backend routes, a database setup, and a user-facing interface.

---

## Features

- Node.js + Express web server
- EJS templates for views
- SQL scripts to create and seed the database
- Modular routing
- Simple database connection module
- CSS file with pink and white Figma style UI

---

## Prerequisites

Make sure you have the following installed:

- Node.js 
- npm 
- MySQL
- Express

## Database Set Up

```bash
mysql -u 'username' -p < create_db.sql
```

## To run

```bash
node index.js
```

## Run in web browser

```bash
localhost:8000
```
